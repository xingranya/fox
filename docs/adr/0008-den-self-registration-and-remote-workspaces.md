# ADR-0008：单组织自助注册与 Den 远程工作区

- 状态：已接受
- 日期：2026-07-24
- 决策人：Fox
- 影响范围：Den 账号策略、FoxWork 登录、组织、远程工作区、Worker、中文管理面、阶段门和 Wiki 发布
- 补充：[ADR-0007](0007-adopt-openwork-den-control-plane.md)

## 背景

团队只有少量内部员工，可以在公司 Den 入口自助注册。员工不应等待管理员逐个创建账号，也不能因此获得创建第二组织或绕过项目授权的能力。同时，最终产品需要员工登录 FoxWork 后直接进入公司提供的远程工作区，不能把远程运行能力长期留在候选项。

F3.2 已用合成账号证明 Den 的单组织自助注册、第二组织拒绝、桌面交接、MCP、Skills 和共享模型可以内部部署。远程 Worker 的真实自托管链路尚未通过，因此仍属于 F3.3-F3.19 的待实现与待验收范围。

## 决定

1. Den 使用 `single_org` 模式，允许普通员工从公司配置的 Den 入口自助注册；员工不能创建第二组织，不能因注册自动获得 Brand OS 项目权限或业务审批权。
2. 员工在 FoxWork 中完成注册或登录。FoxWork 不增加 Brand OS 账号、第二套密码、登录码或旧“团队连接”页面。
3. 普通员工首次访问 Brand OS 时，服务依据可信 Den 发行者、唯一公司组织、有效成员关系和稳定 `(issuer, subject)` 建立内部身份映射。该记录不是可登录的第二套账号；邮箱不能作为身份键自动合并或重新绑定账号，项目权限仍由显式成员或团队映射授予。
4. Den 负责远程工作区与远程 Worker 的创建、分配、运行策略、状态和撤销。FoxWork 统一展示连接、运行和诊断状态。
5. FoxWork、Den 员工页面和 Den 后台管理员网页全部使用自然简体中文，不提供语言切换、英文回退、内部键名或开发占位文案。
6. Den Worker 文件系统、OpenWork/OpenCode Session、缓存和模型摘要都是运行态，不保存品牌项目正式状态或原件唯一副本。
7. Brand OS Outbox/Worker 继续负责资料准入、多媒体解析和派生 Artifact；它与 Den 远程 Worker 是不同职责，不能共享正式写权限。
8. 远程 Worker 不能挂载、浏览或控制员工电脑。所有本机文件、终端和桌面动作必须由当前设备上的 FoxWork 获取员工授权并通过本机桥接执行。
9. 测试 Mac 使用合成数据运行 Den、远程 Worker、Brand OS Service 及临时存储；临时账号、密钥和连接凭据不得进入仓库、文档、日志或构建产物。
10. Wiki 只在大阶段目标完成并通过对应阶段门后统一更新。F3.3-F3.18 的中间实现和普通 SPEC 校准只提交仓库文档；Phase 3 Wiki 在 F3.19 通过后同步。

## F3.3-F3.19 影响

- F3.3：增加单组织自助注册配置、第二组织拒绝、远程工作区/Worker 部署、健康、隔离、升级和回滚。
- F3.4：完成 FoxWork 注册/登录、Den 员工页面、后台管理员网页和完整错误状态中文化。
- F3.5：把 Phase 2 的预登记身份基线升级为可信 Den 成员首次登录建档，不增加第二套登录或密码。
- F3.6：建立 Den 组织、团队、远程工作区与 Brand OS 项目工作区的显式映射和撤权对账。
- F3.11：同时验证本机桥接与远程 Worker 权限边界。
- F3.19：把自助注册、全中文管理面、远程工作区、Worker 故障/恢复和凭据产物扫描纳入联网产品门。

## 验收

- 普通员工可自助注册并只加入唯一公司组织，无法创建第二组织；登录一次即可回到 FoxWork。
- 首次访问 Brand OS 无需管理员预建第二套账号；服务只建立内部身份映射，未获得显式项目授权时显示中文无权限状态。
- 员工只看到获授权的 Den 远程工作区和 Brand OS 项目；撤权后 FoxWork、Worker、MCP、Skills 和模型访问一致失效。
- 远程工作区可创建、连接、停止、恢复和清理；失败状态在 FoxWork 中以自然简体中文显示。
- Den 后台管理员网页的账号、组织、团队、工作区、模型、MCP、Skills、插件、设置和错误状态全部为简体中文。
- 清空 Worker 运行数据不会损坏 PostgreSQL 正式事件、审批、投影或 S3 原件。
- 服务器和远程 Worker 无法绕过 FoxWork 本机授权访问员工电脑。
- F3.19 未通过前不接入真实鸿喜达资料，也不更新阶段 Wiki。
