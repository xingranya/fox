  - 验收：授权、保密级别和处理方清单完整；团队文件协作不自动合线。

- [ ] **F4.2：验证多客户端并发、幂等和状态一致性**

- [ ] **F4.9：完成团队生产 Go/延长/No-Go 与归档**

  - 验收：Fox 明确批准范围、责任人、SLO 和限制；未通过时延长或停止，不虚报上线。
