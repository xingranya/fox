## 使用限制

1. `DEMO-001` 含高敏感身份与财务字段，任何模型、MCP、Skill、Dify 或外部工作流默认不得读取其正文。
2. `DEMO-002` 与 `DEMO-003` 内容相关只能记为待人工确认的关系假设，不能自动认定为同一版本、正式歌词或已批准交付物。
3. `DEMO-003` 的模型转写未经人工校对，不得作为正式原话证据，也不得覆盖人工文稿。
4. `DEMO-004` 属于其他品牌的会议样本，不得迁移为鸿日事实、决定、约束、行动或截止时间。
5. `DEMO-005` 可用于演示鸿日资料解析和候选提取，但任何目标、策略、费用或计划内容在 Fox 确认前均不得进入正式状态。
6. 所有 AI 处理结果只能生成工作层记录或 Proposal；工具执行权限不等于业务批准。
